Looping

Ruby

  • Loop until break
i = 0
loop do 
    i += 1
    next if i == 3
    break if i == 10
end
  • Iterate over each element
temp = [1,2,3]
temp.each do |elem|
    #do something with elem
end
  • While loop
i = 0
while i < 10
    #do something
    i += 1
end
  • Until loop
i = 0
until i > 10
    #do something
    i += 1
end
  • For loop
for val in [1,2,3,4,5]
    #do something with val
end

Javascript

  • Iterate over i
for (var i = 0; i < 10; i++) {
    //do something
}
  • Iterate over each key
var temp = [1,2,3]
for (key in temp) {
    //do something with key
}
