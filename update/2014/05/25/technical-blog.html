What is Rails

Overview

Ruby on Rails, or simply - Rails, is an open source web framework built using the language Ruby. It is a full stack framework (front-end to back-end) and as a result, it features a routing system that is independent of the web server it is hosted on.

Like other web-frameworks, RoR uses the Model-View-Controller pattern to organize application programming. Here is a basic description of what each part is responsible of: - Model: Maps to a table in the server's database - Controller: This responds to external requests to the server and determines which view file to render. The controller is also responsible for getting information from the Model to display at the view. - View: This is basically what the user sees on his or her browser.

History

This was originally created by David Hansson from his work on Basecamp. It was released July 2004.

Philosophy & Design

Rails is intended to emphasize Convention over Configuration, and the Dont Repeat Yourself Principle. It is important to remember to have "Fat models and skinny controllers" when creating your web application.
