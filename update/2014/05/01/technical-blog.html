Ruby Classes

Using Ruby objects is an interesting way to look at programming. Instead of writing down each piece of code that you then execute upon page load, you can think of creating little robot objects that will tackle the work for you.

To better visualize this, I will create 2 usb sticks. Each of them has their own name, data, & capacity. They will also have an instance method called "Transfer" where they transfer their data to another USB stick.

1) Create the USBStick class

class USBStick
    def initialize(name = 'no-name usb stick', data, capacity = 2)
        @name = name
        @data = data
        @capacity = capacity
    end
    def name
        @name
    end
    def data
        @data
    end
    def capacity
        @capacity
    end
end

2) Create the transfer method

class USBStick
   def initialize(name = 'no-name usb stick', data = nil, capacity = 2)
        @name = name
        @data = [data]
        @capacity = capacity
    end
    def name
        @name
    end
    def data(data = nil)
        if (data != nil)
            @data << data.to_s
        else
            @data
        end
    end
    def capacity
        @capacity
    end
    def transfer(usb)
        if (usb.data.length < usb.capacity)
            usb.data(@data)
            p "Succesfully transferred data to #{usb.name}"
        else
            p "Not enough capacity on #{usb.name}. #{usb.data.length}/#{usb.capacity}"
        end
    end
end

4) Lets create 2 usb stick instances and transfer data from one to another

red_stick = USBStick.new('RED', "Random String", 2)

blue_stick = USBStick.new('BLUE', "Blue is not my favorite color", 2)

red_stick.data
# => ["Random String"]

blue_stick.data
# => ["Blue is not my favorite color"]

red_stick.transfer(blue_stick)
# => "Succesfully transferred data to BLUE"

blue_stick.data
# => ["Blue is not my favorite color", "Random String"]

Hope this all makes sense!
