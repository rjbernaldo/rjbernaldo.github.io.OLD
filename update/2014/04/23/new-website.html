New Github Page

So I've redesigned my GitHub blog using something called Jekyll. It is sort of a CMS, minus the complications of maintaining a database. How? Upon running the command jekyll build over on CLI, Jekyll takes your website's local folder and converts all the markdowns to their appropriate HTML structure to be served on web servers.

The end result is a static site that is easily maintainble, and can be hosted anywhere (even on github, like this website right now!)
