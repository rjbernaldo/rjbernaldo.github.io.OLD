Enumerable: Group By

Group By is a very interesting method to help group data using specific rules. It basically groups the output according to what elements fit a specific declared rule. As an example:

1) Group the collection of integers together using their result + 1 as their key

collection = [1,2,3,4,5]
collection.group_by { |i| i + 1}
 => {2=>[1], 3=>[2], 4=>[3], 5=>[4], 6=>[5]}

2) Group the collection according to % 2

collection = [1,2,3,4,5]
collection.group_by { |i| i % 2}
 => {1=>[1, 3, 5], 0=>[2, 4]}

3) Group the collection according to if element == 3

collection = [1,2,3,4,5]
collection.group_by { |i| i == 3}
 => {false=>[1, 2, 4, 5], true=>[3]}

Hope I was able to shed some light on a new method!
